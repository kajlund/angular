The ng-switch directive

The ng-switch is a more powerful version of the ng-show/ng-hide directives.

    var myVar = someExpression;
    switch (myVar) {
        case 'A':
            // myVar = 'A'
        case 'B':
            // myVar = 'B'
        default:
           // if nothing else
    }
