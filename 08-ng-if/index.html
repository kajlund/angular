The ng-if directive

The ng-if directive is used for adding and removing DOM elements based on a truthy/falsy condition.

Any expression that can be evaluated as truthy or falsy is fine. I.e. You don't need to provide boolean expressions only to ng-show and ng-hide

ng-if actually adds/removes DOM elements on the page. It does this by commenting them out instead of actually removing them. The reason is that Angular would otherwise need to keep track of where they should be added back.
