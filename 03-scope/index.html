Sample 3 - Scope Binding

Angular architecture entails binding controller data to a view. This is done via the built-in $scope object.

A $scope object is a JavaScript object with all sorts of goodies like change detection for syncing our models and views.This happens via a thing called the digest cycle.

At this point we just need to know that we can add properties to the $scope object when we want to expose them to our views.
