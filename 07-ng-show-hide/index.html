The ng-show and ng-hide directives

The ng-show and ng-hide directives are used for showing and hiding DOM elements based on some condition.

    var el = document.getElementById('myDiv');
    if (myCondition) {
        el.show();
    } else {
        el.hide();
    }

Any expression that can be evaluated as truthy or falsy is fine. I.e. You don't need to provide boolean expressions only to ng-show and ng-hide

Show/hide does not remove DOM elements but sets and clears class ng-hide which in turn has a style rule of display: none !important;
