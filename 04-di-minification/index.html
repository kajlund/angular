Dependency Injection and Minification

By adding the dependencies we need as function params, AngularJS will insert them at runtime. This is called dependency injection

Minifying your code would break dependency injection so we need to use the $inject property and provide it with an array of our dependencies as strings. Strings as opposed to function params names will not be minified and Angular can use these strings and map them against the minified function param names.
